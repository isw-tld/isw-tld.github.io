Gegevens laten zien

De theorie van informatica actief kun je eventueel nalezen via deze link

Voor dat we leren hoe we de gegevens uitgebreid kunnen gebruiken gaan we eerst leren hoe we ze kunnen de gegevens op de website kunnen laten zien. Dit doen we door gebruik te maken van de functie echo zoals we geleerd hebben bij PHP. De simpelste manier om alles te laten zien is als volgt: 


while (list($id, $soort, $kleur, $aantal) = mysqli_fetch_row($result)) {
  echo "$id $soort $kleur $aantal <br>";
}

Het volgende gebeurt in de code

  1. We gebruiken een while-lus om alle rijen langs te gaan.
  2. Elke rij slaan we op in een lijst, met daarin alle kolommen die we in de query hebben geselecteerd. Dit doen we met de functie list() en mysqli_fetch_row()
  3. De volgorde van de kolommen in de lijst, is hetzelfde als bij de query
  4. Vervolgens laten we met echo de gegevens zien die we belangrijk vinden.

Opdracht

Zorg ervoor dat je alle gegevens van de gebruikers op je website kan zien.
