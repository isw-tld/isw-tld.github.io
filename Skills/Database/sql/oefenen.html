Oefenen en Overzicht

Voordat we de laatste opdrachten gaan maken, krijg je eerst nog een overzicht van hoe alle keywords samen werken. Deze is op de standaard placeholder methode genoteerd. 


      SELECT <kolommen>
      FROM <tabellen>
      WHERE <voorwaardes>
      GROUP BY <kolommen>
      ORDER BY <kolommen> <ASC/DESC>

Een voorwaarde bestaat uit <kolom> <operator> <waarde>.
