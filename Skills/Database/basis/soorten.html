Databases

Een database is plek waar je informatie kan opslaan, bewerken en opzoeken. Als je deze beschrijving leest kun je zien dat Excel, misschien wel de meest bekende database is die je kent. In Excel gebruik je de cellen om gegevens in op te slaan, deze kun je veranderen als er iets anders is geworden en wanneer je ergens naar zoekt kun je het gemakkelijk vinden met de zoek functie. Er zijn mensen die Excel gebruiken als wachtwoord beheerder, adresboekje, spaaroverzicht, of bijvoorbeeld het bijhouden van mensen die ergens bij horen, bijvoorbeeld voor een feestje. Wanneer je gaat programmeren wil je soms ook gegevens bijhouden, denk aan high-scores, artikelen, gebruikers, of andere informatie die met elkaar verbonden is. Deze laatste eis maat de database compleet, de data (gegevens) in een database moet gestructureerd zijn. Er zijn verschillende manieren om de data te structureren, hier komen verschillende soorten databases in beeld.

Soorten databases

Er zijn vrij veel verschillende type databases die gebruikt worden. Het soort waar wij het meest naar gaan kijken heet Relationele database. Dit type databases komt uit de jaren ’80 en wordt nog veel gebruikt. Relationele databases gebruiken tabellen die met elkaar verbonden worden om zo de data met elkaar te verbinden. Andere type databases kun je vinden in de tabel hieronder

Database Omschrijving Gebruik
Gedistributeerde database De bestanden voor deze database zijn verspreid over verschillende computers. Door de database op verschillende plekken te zetten ben je minder kwetsbaar voor dataverlies en netwerk falen.
NoSQL database NoSQL is de term voor niet-relationele databases. Deze data heeft weinig structuur. Deze manier van werken wordt gebruikt op het moment dat de structuur van de data complex is, zoals bijvoorbeeld big-data
Flat-structure database Deze manier van databases maken gebruikt tekst om gegevens te ordenen. Meestal wordt er gebruikt gemaakt van keys, namen, met daarbij values, de waarde die erbij hoort. Dit soort databases zijn erg simpel om te maken en op te slaan, het is voor een computer ook niet moeilijk om te lezen, maar je moet alle informatie gelijk verwerken waardoor het niet zo efficiënt is.
