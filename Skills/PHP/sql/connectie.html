Connectie maken

De theorie van informatica actief kun je eventueel nalezen via deze link

Nu zijn we klaar om de database te verbinden met alle kennis die we tot nu toe opgedaan hebben. Eerst moeten we verbinding maken met PHPMyAdmin. Dit doe je door in een nieuw php bestand genaamd server.php de volgende gegevens in te vullen 


<?php
  $db="naam van database";
  $server="localhost";
  $naam="root";
  $ww="usbw";
?>
De enige die je daadwerkelijk moet vervangen is de naam van de database, deze verander je naar de database die je gaat gebruiken, bijvoorbeeld de bloemist database uit het de database theorie.

Vervolgens kun je de verbinding in jouw website verbinden door de volgende code te gebruiken: 


<?php
  include "server.php";
  $sql = mysqli_connect($server,$naam,$ww,$db) or die("verbinden mislukt");
?>
Nadat je deze code hebt toegevoegd kun je queries uit voeren op de database, Tenzij je natuurlijk de melding verbinding mislukt krijgt. Als je deze krijgt moet je even goed kijken of de naam van de database en de andere gegevens goed zijn ingevuld. Weet je het echt niet meer, vraag dan altijd om hulp!
