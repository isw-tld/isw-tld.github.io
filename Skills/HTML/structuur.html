Structuur

Je hebt nu de basis van de HTML pagina onder de knie, tijd om verder te kijken naar de inhoud van de website. Voordat je een website maakt is het verstandig om een idee te hebben van hoe je zou willen dat je website eruit ziet. Voor elke app die je maakt, is het verstandig om een ontwerp te maken. Het kan simpelweg gemaakt worden in een tekenprogramma, of zelfs met de hand als je dat fijner vindt.

Hiernaast zie je een foto van een voorbeeld layout die je zou kunnen gebruiken, maar er zijn natuurlijk ook andere indelingen mogelijk. HTML heeft een aantal tags die gebruikt kunnen worden om de opmaak van jouw website goed te krijgen. De volgende tags zijn erg handig:

<header> De kop van de website, hierin kan bijvoorbeeld een logo.
<footer> De onderkant van de website, hierin staan vaak contact gegevens.
<aside> Een zijbalk, deze is bijvoorbeeld te gebruiken bij een menu aan de zijkant.
<nav> De menubalk van de website, hiermee navigeer je over de site.
<article> Gebruikt om een stuk onafhankelijke inhoud weer te geven, denk bijvoorbeeld aan een nieuws artikel.
<section> Gebruikt voor een stuk bij elkaar horende informatie op de website.
<details> Een stuk informatie die verstopt kan worden door de gebruiker, bijvoorbeeld voor een begrippen lijst.
<summary> De titel van het detail blokje.

Teken in paint, of met draw.io een schets van hoe je wilt dat jouw website opgebouwd is. Voeg deze met behulp van de img tag toe aan jouw website.
