Sorteren

Van naturen is de database gesorteerd op volgorde van wanneer het is toegevoegd. Alles wat je toevoegt komt namelijk onderaan te staan. Je kunt de volgorde veranderen als je dat nodig hebt. Hiervoor voegen we het keyword ORDER BY toe helemaal aan het einde van de query. 


        SELECT <kolommen>
        FROM <tabellen>
        ORDER BY <kolommen> ASC/DESC

Je kunt meerdere kolommen gebruiken om binnen de data ook nog te sorteren. Let op! Het maakt uit op welke volgorde je dit doet. ASC wordt gebruikt om de waardes op oplopende volgorde te zetten, dit is dus vanaf 0-9 en daarna A-Z. Als je het andersom wilt dan kun je gebruik maken van DESC, dit staat voor aflopend. Achter elke kolom kun je ASC of DESC zetten zodat je het volledig kan sorteren naar jouw wens.

  2. Verander de query om eerst het geslacht en daarna de namen te sorteren, wat is het verschil?
  3. Wat kun je concluderen over de volgorde van ORDER BY?
