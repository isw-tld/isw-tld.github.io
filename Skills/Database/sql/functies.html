Functies

We hebben nu de eerste basis queries gemaakt. Nu wordt het tijd om wat meer functionaliteit toe te voegen. hiervoor heb je zelf al gezocht naar de jongste en oudste leerling, maar als er 1000 leerlingen op een school zijn gaat dit wel even duren, gelukkig kan het ook makkelijker. Bij het SELECT keyword kun je ook gebruik maken van functies een voorbeeld hiervoor zou zijn: 


        SELECT MAX(geboortedatum) 
        FROM Leerlingen
In deze query krijg je één resultaat terug met alleen maar de grootste geboortedatum die gevonden kon worden. Dit betekent in dit geval, de jongste leerling. Als je dit op tekst zou uitvoeren dan krijg je het resultaat wat als laatste zou voorkomen in het alfabet, en op nummers natuurlijk het grootste getal. Je kunt dit ook omdraaien door MIN(<kolom>) te gebruiken. Andere functies die je moet kennen zijn:

Functie Omschrijving Voorbeeld
SUM(<kolom>) Telt alle waardes uit de kolom bij elkaar op. SELECT SUM(boete)
FROM boetes
AVG(<kolom>) Berekent het gemiddelde van de waardes uit de kolom SELECT AVG(cijfer) 
FROM cijfers
COUNT(<kolom>) Telt de hoeveelheid waardes uit de kolom SELECT COUNT(*)
FROM leerlingen

  1. Importeer de database van hoofdstuk 2 in PHPMyAdmin. Dit is een andere database die makkelijker gebruik maakt van de functies die we in deze paragraaf leren.
  2. Schrijf een query om de totale aantal bloemen te bepalen.

Groeperen

Count kan heel handig zijn om bijvoorbeeld te bekijken hoeveel leerlingen een bepaalde eigenschap hebben. Je kunt bijvoorbeeld kijken hoeveel leerlingen er per geslacht zijn. Hiervoor hebben we alleen nog een ander keyword nodig genaamd GROUP BY. Dit keyword gebruik je om een bepaalde kolom samen te voegen hiermee kun je bijvoorbeeld een functie maken zoals: 


        SELECT COUNT(*), geslacht
        FROM leerlingen
        GROUP BY geslacht

Dubbelen weg halen

Wat je doet bij GROUP BY, is het groeperen van alle unieke waardes uit de kolom. Voor de bloemen uit opdracht 8 kun je bijvoorbeeld alle kleuren groeperen, of bijvoorbeeld alle soorten. Er is ook een manier om alleen de unieke waardes te vinden van een kolom. Dit doe je door een keyword toe te voegen aan SELECT, namelijk DISTINCT. In het voorbeeld kun je zien dat deze niet op dezelfde manier gebruikt wordt als de andere functies uit deze paragraaf. 

        SELECT DISTINCT kleur
        FROM bloemen
Het resultaat wat hier uit komt is een lijst met daarin 1x alle kleuren die voorkomt in de kolom kleur.

  2. Schrijf een query om alle soorten te vinden van de bloemen in de database.
  3. Schrijf een query om het gemiddelde aantal bloemen te vinden van elke geregistreerde bloem uit de database.
