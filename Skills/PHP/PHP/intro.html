Introductie PHP

Om PHP te gebruiken moet je de USBwebserver gebruiken en de websites die je maakt opslaan in de map die staat bij de instellingen in root dir. Vervolgens moet je daar een bestand hebben die als extensie .php heeft. En als laatste moet je ervoor zorgen dat je de website opstart via de localhost knop op de USBwebserver. Het werkt niet via notepad++.

Formulieren

Om gegevens naar PHP te sturen maak je gebruik van een formulier van HTML. In de standaard pagina zet je dan het volgende: 


<!DOCTYPE html>
<html>
  <head>
    <title>Eerste Formulier</title>
  </head>
  <body>
    <h1>Formulier invullen</h1>
    <form action="" method="post">
      <label for="vnaam">Voornaam:</label>

      <input type="text" id="vnaam" name="vnaam"><br>
      <label for="anaam">Achternaam:</label>

      <input type="text" id="anaam" name="anaam">
      <input type="submit" value="Verstuur">
    </form>
  </body>
</html>
In dit stuk zie je het formulier. Dit formulier wordt geopend met <form action="" method="post">. Wat hierbij van belang is, is dat je weet dat de actie vertelt waar de inhoud naartoe wordt gestuurd. Als je het leeg laat zoals hier, dan wordt het gestuurd naar de huidige pagina. De methode wordt gebruikt om aan te geven hoe het wordt verstuurd. Post betekent dat we het versturen, dit is de enige die je nu moet kennen.

In het formulier maak je daarna gebruik van de input tag, om velden te maken voor de input. Je hebt verschillende soorten zoals text en submit zoals in het voorbeeld. Er zijn er ook nog meer deze kun je vinden op W3Schools.
De submit knop, is nodig voor het versturen van de informatie. Nadat het formulier verstuurd is, kun je de gegevens in PHP ophalen. Hiervoor moeten we eerst een PHP blok maken in de code. Het PHP blok maak je door middel van <?php ... ?>. Op de plek van de puntjes kun je nu de code neerzetten. 


<?php
  // haal de inhoud uit het formulier
  if (isset($_POST["vnaam"])) {
    $naam = $_POST["vnaam"];
    echo $naam;
  } else {
    echo "Geen voornaam opgegeven.";
  }
?>
In het voorbeeld hierboven kun je nu zien hoe we de gegevens ophalen uit het formulier. De code werkt als volgt:
  1. Kijk of er een POST is gedaan voor een input genaamd vnaam.
  2. Als deze er is, dan slaan we de POST waarde voor vnaam op in de variabele naam en laten het zien op het scherm.
  3. Als deze er niet is, dan laten we op het scherm zien dat er geen voornaam gegeven is.
Als het goed is moet je de gewone constructie van if en else gewoon nog kennen. Als dit niet het geval is, maak je nog niet te veel zorgen we gaan het nog eens kort herhalen later, voor nu kun je gewoon gebruik maken van het voorbeeld zoals hij gegeven is. We versturen niet altijd het formulier, bijvoorbeeld de eerste keer als we de website laden. Om er dus voor te zorgen dat we geen foutmelding krijgen gebruiken we de if(isset($_POST["vnaam"])) hiermee kijk je of de post waarde wel bestaat.

Echo

Als laatste stukje voor deze theorie is de echo. De echo gebruik je om stukjes tekst, of gegevens te laten zien op het scherm. Alle andere code wordt namelijk verstopt. Als je een echo gebruikt, zorg er dan voor dat de PHP code nog in het <body> blok staat van de HTML, hierdoor kan je ook gebruiken van HTML en CSS in de code. Vergeet niet om alle regels die je schrijft te laten eindigen op ; anders snapt PHP het niet. De enige uitzondering is alle regels die je kan sluiten met accolade openen, hiervoor zie je bijvoorbeeld de if deze eindigt met een {.
