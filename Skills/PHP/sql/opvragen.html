Query's uitvoeren

De theorie van informatica actief kun je eventueel nalezen via deze link

De eerste query die we gaan doen is met de database van de bloemist van het vorige onderdeel. We doen een query om alle gegevens te laten zien uit de tabel bloemen. Hiervoor gebruiken we natuurlijk de query: "SELECT * FROM bloemen". De volledige code ziet er dan zo uit: 


<?php
include "server.php";
$sql = mysqli_connect($server,$naam,$ww,$db) or die("verbinden mislukt");
$result = mysqli_query($sql,"SELECT * FROM bloemen") or die("Alle bloemen ophalen is mislukt")
mysqli_close($sql) or die("verbinden sluiten mislukt");
?>

Hier zie je het volgende gebeuren:

  1. De verbinding met de server wordt gemaakt.
  2. De query voor SELECT * FROM bloemen wordt uitgevoerd op de database. Als de query mislukt, krijg je de error “Alle bloemen ophalen is mislukt”.
  3. Als we alle queries hebben gedaan dan sluiten we de verbinding weer af.
  4. Dan wordt er voor elke rij in de database, tijdelijk een lijstje gemaakt met alle kolommen die we hebben opgevraagd. Deze waardes laten we zien achter elkaar gevolgd door een enter.
Let erop dat bij het openen, uitvoeren van queries en het afsluiten van de database je altijd een or die gebruikt om foutmeldingen op te vangen. Als je dit niet doet zou een hacker gegevens over jouw database kunnen verzamelen.

Opdracht

Maak de query die je nodig hebt om alle gegevens van de gebruikers van jouw database te laten zien. Vul de query op de juiste plek in de code. Het resultaat zou moeten zijn dat je niet de foutmelding krijgt!
