MacOS (en Linux)

De USBwebserver draait zoals je weet niet op Mac computers. Om ervoor te zorgen dat je toch thuis aan je website kan werken heb ik gezocht naar een alternatief en vond deze optie.

Het programma

Het programma wat je kunt gebruiken heet MAMP. Dit programma kun je gratis gebruiken (MAMP PRO is betaald). Het programma is een webserver die net als USBWebserver PHP en MySQL kan draaien. Daarmee heb je voldoende om de opdrachten van de website te maken en het PO te maken.

Installatie

Om MAMP te installeren ga je naar de MAMP website. Als je hier downloadt dan krijg je MAMP en MAMP PRO tegelijk. Je kunt als je MAMP wilt opstarten via spotlight (command spatie) kiezen voor de MAMP-versie in plaats van MAMP PRO. Alternatief kun je ook gelijk MAMP pro de√Ønstalleren op de volgende manier:

  1. Start MAMP PRO op.
  2. Klik boven in het scherm op MAMP PRO.

  4. Volg de de-installatie instructies.
Als je MAMP PRO weg hebt gehaald, heb je alleen nog de gratis versie van MAMP op je Mac staan.

Databases

Als je een database wilt toevoegen dan kun je PHPMyAdmin gebruiken. De standaard inlog gegevens van MAMP zijn root voor zowel het wachtwoord als de gebruikersnaam. De link hiervoor is: http://localhost:8888/phpMyAdmin/?lang=en

Werken op school en thuis

Als je van computer wisselt, dan moet je ook de database mee nemen, zorg ervoor dat je de databases die je zelf maakt exporteert en op OneDrive zet. Vervolgens kun je het dan meenemen naar de nieuwe locatie.
