USBwebserver

USBwebserver is een programma wat op de specifieke computer staat waar je op werkt. Als jij dus dingen opslaat op de computer dan staat het ALLEEN op die computer en kan je er niet bij vanaf een andere computer. Om er toch voor te zorgen dat je bij je bestanden kan moet je ervoor zorgen dat je elke les jouw spullen netjes opstuurt naar jezelf en degene waarmee je werkt.

Website opslaan

Website opslaan Jouw website kan op 1 van de volgende plekken staan:

  1. In OneDrive, zorg er dan voor dat:
    1. De computer OneDrive heeft.
    2. Dat de USBwebserver in de instellingen de OneDrive map als root map heeft.
    3. Let op! het werkt niet goed om via OneDrive te delen en samen te werken. De database gaat dan stuk (ja, dat was stressvol)
  2. In de root map van de USBwebserver. Deze versie is iets moeilijker, namelijk, je kan er alleen bij door in de USBwebserver te klikken op de knop root dir. Zorg ervoor dat jij jouw website opslaat in een mapje die jullie naam, leerling nummer of project naam (bijvoorbeeld “schoenenwinkel”) heeft en dat je dit mapje dan telkens naar jou en je groepsgenoot opstuurt.
Een paar belangrijke dingen:
  1. Je moet je website opstarten vanuit de knop localhost, het kan NIET gestart worden vanuit notepad++ zoals je in de vierde klas kon doen.
  2. Zorg ervoor dat Apache en Mysql allebei een vinkje hebben, anders werkt je website niet.
  3. Zorg ervoor dat alle bestanden op de goede plek staan.
Als je er niet uit komt kan je altijd aan je docent om hulp vragen.
