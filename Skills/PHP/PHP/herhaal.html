Soms wil je een stukje code meerdere keren uitvoeren. Je zou kunnen denken dat je dan gewoon de code een paar keer achter elkaar plaatst, maar dat is natuurlijk niet zo mooi en niet handig als je een stukje wilt veranderen. Er zijn twee opties: 1. Je weet hoe vaak je de code wilt uitvoeren. Bijvoorbeeld 5 keer of voor de lengte van een woord. 2. Je weet niet hoe vaak je de code wilt uitvoeren. Bijvoorbeeld als je niet weet hoe je de lengte van iets krijgt. Dit heb je als je waardes uit jouw database haalt.

For-lus

Als je een for lus gaat maken dan gebruik je de volgende code: for($teller = 0; $teller < $aantal; $teller++) { echo "Deze zin is nu $teller keer geprint."; } Wat je hier ziet is dat de for lus tussen de haakjes 3 dingen heeft staan: 1. Teller. Deze gebruik je om bij te houden hoever je bent. Je begint vaak bij 0 maar je kan ook met een ander nummer beginnen. 2. Voorwaarde. Deze gebruik je om aan te geven hoelang je door wilt gaan. Bij het voorbeeld als je $aantal = 5 doet dan voer je de code 5 keer uit. 3. Optellen. Om ervoor te zorgen dat de teller ook daadwerkelijk groter wordt moet je bij het optellen ervoor zorgen dat je de teller groter maakt. Dit kan in stapjes van 1 of 2, je kan ook aftellen als je in plaats van de plussen twee minnen gebruikt. Daarna gebruik je weer de accolades om aangegeven welke code hoort bij de lus.

While-lus

Als je niet weet hoe vaak je code moet uitvoeren of als je geen teller gebruikt dan kan je beter gebruik maken van een while lus. Hier zal je tijdens je praktische opdracht. De while lus maak je als volgt: while(list($voornaam, $achternaam) = mysqli_fetch_row($klantenNamen) ){ echo "Klant: $voornaam $achternaam bestaat ook"; } Ook hier zie je weer dat alles binnen de accolades de opdracht is die bij de while lus hoort en net als bij de if en for heb je tussen gewone haakjes de voorwaarde staan om de while lus uit te voeren. In de voorwaarde zie je de opdracht: list($voornaam, $achternaam) = mysqli_fetch_row($klantenNamen). Dit wordt uitgelegd in het stukje over gegevens laten zien.

Foreach-lus

Ook als je een array hebt, kun je een speciale for loop gebruiken genaamd de for each loop om alle waardes te laten zien: "35", "Ben"=>"37", "Joe"=>"43"); foreach($age as $x => $val) { echo "$x = $val
"; } ?> Deze speciale array die je hier ziet gebruikt de naam van de persoon als sleutel, en de leeftijd als waarde hiermee kun je bijvoorbeeld $age["Peter"] gebruiken om de leeftijd van Peter te krijgen. Dit type array is handig als je bijvoorbeeld een winkelwagen, to do lijst, of een high score wilt bijhouden. De $x die je ziet in de foreach houdt op dit moment de naam van de personen, de $val onthoudt de leeftijd per persoon. Je hoeft de waarde voor $x of $val niet te gebruiken, je kunt ook een van de twee gebruiken!
