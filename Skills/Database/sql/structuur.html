Queries

SQL is een taal die we gebruiken voor database vragen, het is een acroniem voor Structured Query Language. Hieruit kun je lezen dat het een gestructureerde taal is die bedoelt is voor query's. Een query is een vraag die je stelt aan de database. Als je een query uitvoert dan zijn er drie opties:

  1. Je krijgt de bijbehorende rijen terug
  2. Je krijgt niets terug, er is geen resultaat
  3. De query runt niet, je hebt een foutmelding
Afhankelijk van het resultaat wat je terug krijgt kun je concluderen of je de query goed hebt gemaakt, maar nu lopen we te ver vooruit.

Structuur

De structuur die gebruikt wordt in de basis van een query ziet er als volgt uit:


        SELECT <kolommen>
        FROM <tabellen>

Wat je hier ziet is dat we beginnen met het woord SELECT, dit is een keyword om te vertellen wat we willen weten. Daarachter staat <kolommen> dit is een placeholder. De haakjes en het woord kolommen schrijf je niet op, in plaats daarvan zeg je bijvoorbeeld naam of geboortedatum om deze kolommen van de leerlingen op te vragen. Het is belangrijk om te weten dat SQL niet hoofdletter gevoelig is, het maakt dus niet uit of je Naam, NAAM of naam gebruikt. Het zelfde geld hier voor de tabellen. We gebruiken hiervoor het keyword FROM om te vertellen uit welke tabel de kolommen gehaald moeten worden, hierbij vul je de naam van de tabel in. Tot nu toe hebben we nog niet officieel een naam gegeven aan onze tabellen, zorg ervoor dat je naam beschrijft wat het is, zitten er leerlingen in de tabel, dan noem je het leerlingen, is het voor de ouders, dan noem je het ouders. Maak het niet te moeilijk.

Wanneer je meerdere kolommen wilt zien maak je gebruik van een comma tussen de kolommen in. Hier een volledig voorbeeld voor het opvragen van alle namen en e-mail adressen van de leerlingen.
