Jullie database!

Kies een van de opdrachten voor je website. Deze kun je vinden op magister. Maak vervolgens in het groepje een database. Hierbij lever je het volgende in:

  • Een Word bestand met daarin
    • De namen van alle leerlingen in de groep.
    • De opdracht die jullie gekozen hebben met een naam voor de website.
    • Een overzicht van wie welke query heeft gemaakt uit het SQL bestand.
    • Stroken diagram met het overzicht van de tabellen, beschrijf ook hoe je hem in de juiste normaal vorm hebt gezet.
    • Beschrijving welke andere database soort er eventueel gebruikt had kunnen worden.
    • Voor elke kolom in de gebruikers tabel een beschrijving waarom deze kolom voor jullie van belang is.
  • Een SQL bestand met daarin
    • De query voor het maken van een database.
    • De query's voor het maken van de tabellen in de database, uitgerust met natuurlijk de juiste sleutels.
    • De query's voor het invoegen van minstens 5 rijen voor elke tabel.
    • Let erop dat je de query's op de juiste volgorde zet.
