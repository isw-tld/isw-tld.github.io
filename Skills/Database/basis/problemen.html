Problemen met USBWebserver

De USBwebserver is een programma wat ervoor zorgt dat jouw website kan werken. Een paar belangrijke dingen:

  1. Je moet je website opstarten vanuit de knop localhost, je kunt het NIET gestart worden vanuit notepad++ of Visual Studio Code doen zoals je in de vierde klas kon doen.
  2. Zorg ervoor dat Apache en Mysql allebei een vinkje hebben, anders werkt je website niet.
  3. Zorg ervoor dat alle bestanden op de goede plek staan. Dit betekent dat alle bestanden in jouw root map staan. Als je er niet uit komt kan je altijd aan je docent om hulp vragen.

Website werkt niet

Soms komt het voor dat de website niet werkt. Je hebt bijvoorbeeld een uitroepteken staan bij de apache server in de USBwebserver. Om dit op te lossen is het van belang dat je kijkt naar de instellingen. Hierin staat een invulveld voor de root map. Deze map moet bestaan.
Voor sommige computers is de root map ingesteld op OneDrive. Je ziet dan iets wat begint met O:/. De mappen die hier achter staan moeten dan ook bestaan op jouw OneDrive. Bijvoorbeeld als er staat O:/informatica, dan moet in jouw OneDrive map een mapje zijn die “informatica” heet. Als die er niet is, moet je dat mapje even maken. In die map ga je nu je webpagina’s voor je opdrachten of PO maken.
Niet iedereen heeft toegang tot OneDrive op de computer, verander in dat geval de root map naar {path}/root om apache te laten werken. Nu moet je de veranderingen opslaan en start apache opnieuw op. Als je deze map wilt vinden op de computer dan moet je als je de USBwebserver opstart klikken op root dir.
